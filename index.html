About me

My name is John Dong and I am a Software Development Engineer in Test (SDET) at McAfee. Currently I am residing in Portland, Oregon. I received my Bachelors of Computer Science from Oregon State University with a concentration in Computer Systems in 2014. I have interned at Intel and Elemental Technologies.

I have been in the software industry for almost 3 years. As previous mentioned, I am working at McAfee, specifically with the Orchestrator Cloud (McAfee ePO) security management tool. My team codes, tests and deploys McAfee ePO Cloud. My interest are working with micro-service based architecture and continuous integration in hoping to make one click deployment a reality. My position involves writing Selenium, integration test, Rest API test using the Test NG framework and Maven in Java. My role also includes leading our cloud platform release which involves writing deployment scripts in PowerShell, testing and making sure the code pushed for release passes all quality check-lists.
